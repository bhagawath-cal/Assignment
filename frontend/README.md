# Frontend - VueJS + TypeScript + Quasar

This is the frontend application for the Movie Database Platform.

## Setup

1. Install dependencies:
   ```bash
   npm install
   # or
   pnpm install
   ```

2. Start development server:
   ```bash
   npm run dev
   # or
   pnpm dev
   ```

## npm vs pnpm

- **npm**: The default Node.js package manager
- **pnpm**: A faster, disk-efficient alternative that uses hard links

Both work with this project. Use whichever you prefer.

## Project Structure

- `src/pages/` - Page components (MovieList, MovieDetail)
- `src/components/` - Reusable components
- `src/services/` - API service layer
- `src/router/` - Vue Router configuration
